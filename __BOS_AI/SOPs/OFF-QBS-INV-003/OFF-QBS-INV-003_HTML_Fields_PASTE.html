Printing Customer Invoices in QuickBooks Desktop
How to paste each field into BOS
  1. Click the blue Copy button.
  2. In BOS, open the SOP entity and find the matching field.
  3. Click Source (<>) in the CKEditor toolbar.
  4. Select all (Ctrl+A) and paste (Ctrl+V).
  5. Click Source again to return to the visual view.
field_prerequisites is plain text — each item is copied and entered separately as its own value in BOS.
